1. &nbsp;Clone the repository

```bash
    git clone https://github.com/rks-031/placement-iiit-IE.git
```

2. &nbsp;Install the dependency

```bash
    npm install
```

3. &nbsp; Stat the local UI server

```bash
    npm start
```
